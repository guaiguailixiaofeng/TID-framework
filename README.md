# T-I-D Framework

Code for "A T-I-D Framework for Detecting Critical Transitions in Cancer Progression"

## Files

- `step3_network_construction.maple` - Network construction
- `step3_TID_analysis.maple` - T-I-D framework analysis
- `step4_transition_point_analysis.maple` - Transition point detection
- `step5_hub_gene_identification.maple` - Hub gene identification
- `step6_TIP_transition_analysis.maple` - TIP transition analysis
- `step7_network_rewiring_analysis.maple` - Network rewiring analysis

## Data

TCGA-BLCA dataset from cBioPortal: https://www.cbioportal.org/study/summary?id=blca_tcga_pan_can_atlas_2018
